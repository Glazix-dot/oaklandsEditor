How to use this editor

Add components: click any component in the left palette to drop it onto the canvas.

Wire them together: drag from one port (small square on the side of a component) to another. Outputs are on the right, inputs are on the left.

Wire priority: the real game allows multiple wires into one input port. This editor supports that too — when a port has 2+ wires, a small numbered badge appears on it. Click the badge to open the Wire Priority list and reorder which wire's signal wins (top of the list always wins, like setting which wire "overrides" the others).

Run / Pause: the simulation runs continuously by default. Pause it to freeze all signals while you wire things up, then press Run again.

Interact with inputs: click buttons/switches, drag sliders, use dropdowns — they behave like the real in-game devices (e.g. a Button pulses 10 only while held).

Inspect a component: click it to see its parameters, live input/output values, and a note on exactly how it matches (or approximates) real in-game behavior.

Save your work: use Export to get clean, human-readable JSON you can copy/paste or save anywhere, or Download .json to save a file directly. Use Load file or Import from text to bring a circuit back in.

Note: a few obscure "Other" category components (lasers, conveyors, ownership manager, etc.) don't have fully documented in-game formulas, so they use a clearly-labeled best-effort approximation — check each component's ⓘ note for details.

Export circuit (human-readable JSON)

This is your full circuit — every component, its settings, and every wire (with priority order) — as clean, indented JSON. Copy it anywhere, or download it as a file. You can also hand-edit it and re-import.

Import circuit from text

Paste a previously exported JSON circuit below, then click Import. This replaces everything currently on the canvas.
