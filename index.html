How to use this editor

Add components: click any component in the left palette to drop it onto the canvas.

Wire them together — two ways: click once on a port, then click on another port to connect them (this is the most reliable method on a Mac trackpad, no dragging needed). You can also just drag from one port to another the classic way. Outputs are on the right, inputs are on the left. Press Esc to cancel a pending wire.

Reroute a wire: double-click anywhere along a wire to drop an anchor point, then drag that anchor to route the cable around other components. Right-click (or double-click) an anchor to remove it.

Wire color: pick the exact color for your next wire with the color swatch in the toolbar — works exactly like the in-game Creative Wiring color picker.

Delete a wire: right-click it, or click to select it and press Delete.

Select multiple components: drag a box on empty canvas, or Shift-click components one at a time. Drag any selected component to move the whole group together. Use Duplicate (or Ctrl/Cmd+D) to clone the selection — internal wiring between duplicated components is preserved. Use Delete to remove the whole selection at once.

Wire priority: the real game allows multiple wires into one input port. When a port has 2+ wires, a small numbered badge appears on it — click the badge to reorder which wire's value wins.

Execution order: for components with more than one "event" input (like the Memory Cell's WRITE/RESET, or the Incrementor's ADD/RESET), if two signals arrive on the same tick, whichever wire was connected first is processed first — exactly like in-game. So wire order matters, not just port choice.

Logic gates show their equation: every gate (AND, OR, XOR, XAND, NOT, Greater Than) labels its inputs X and Y and displays its formula right on the node — e.g. the Greater Than Gate shows "Y > X" and outputs Y's value when true.

Hover any port to see exactly what it does, not just its short label.

Run / Pause: the simulation runs continuously by default. Pause it to freeze all signals while you wire things up.

Cost Summary: opens a full breakdown of every component on the canvas with its confirmed in-game price from Alan's AutoLogistics, individual subtotals, and a running grand total. Items without a confirmed price are listed as "n/a" and excluded from the total.

Save your work: use Export for clean, human-readable JSON you can copy/paste or save anywhere, or Download .json for a file. Use Load file or Import from text to bring a circuit back in.

A few "Other" category components (lasers, ownership manager, etc.) are based on the most detailed information available but are still simplified for the editor — check each component's ⓘ note for specifics.
